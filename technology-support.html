GalaxyBrite 360 Setup Guide

This guide provides instructions for using the GalaxyBrite 360 app to configure your pool lights. The app supports both RGBW lights and white lights, allowing you to switch between control platforms and adjust color temperature and brightness settings.

RGBW Light Configuration

In the RGBW Light section, users can select from three popular brands to adapt the RGBW light fixtures to the corresponding control system. After selecting the desired brand, simply press the “Program” button to apply the configuration. The NFC function will initiate, and the app will send the necessary setup information to the pool light product.

White Light Configuration

In the White Light section, you can adjust the color temperature and brightness of the white light fixtures:

  • The top bar and buttons allow you to set the color temperature, with options ranging from 2700K to 6500K.
  • The bottom bar and buttons control brightness levels, offering preset values from 0% to 100%.

Once your desired settings are selected, press the “Program” button to write the configuration to the pool light. The “Read” button allows you to view the current settings in the pool light product, which will be displayed within the app.

Current Setting Display

The Current Setting section shows the active color temperature and brightness (for white lights) or the brand control system (for RGBW lights) currently configured on the pool light.
